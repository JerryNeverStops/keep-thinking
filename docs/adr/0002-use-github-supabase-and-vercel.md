# Use GitHub, Supabase, and Vercel

The application will use GitHub for source control, Supabase for authentication, database records, and uploaded source-text files, and Vercel for hosting the Next.js application. A purely GitHub Pages-based site would be simpler, but it does not fit the required administrator workflow of logging in, uploading PDF or EPUB files, creating anchors, saving notes, and publishing them from the browser.
